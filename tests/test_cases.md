# Casos de Teste

1. Inserção e busca de chave válida  
Verifica se uma chave inserida pode ser recuperada corretamente.

2. Distribuição por distância XOR  
Confirma que a chave é armazenada no nó com menor distância XOR.

3. Busca de chave inexistente  
Busca uma chave que não foi inserida e verifica o retorno de erro.

4. Funcionamento com múltiplos nós  
Garante que a DHT funciona corretamente com vários nós ativos.

5. Validação do cálculo de distância XOR  
Confere se o cálculo XOR retorna valores consistentes.
